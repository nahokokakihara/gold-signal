# 🥇 GOLD シグナル（スマホ版）

スマホに最適化されたゴールド（XAU/USD）売買シグナルWebアプリです。

## 機能

- 📊 **売買シグナル表示** - EMA/RSI/MACDを組み合わせた分析
- 📈 **チャート表示** - ローソク足 + 移動平均線
- 📜 **シグナル履歴** - 過去のシグナルを記録・結果を記入可能
- 🔔 **LINE通知** - シグナル発生時にLINEへ通知
- 📱 **PWA対応** - ホーム画面に追加してアプリのように使用可能

## 使い方

### 1. アプリを開く

`index.html` をスマホのブラウザで開きます。

**ローカルで使う場合：**
PCでローカルサーバーを起動してスマホからアクセス

```bash
# Pythonがある場合
cd gold-signal-mobile
python3 -m http.server 8080
# スマホから http://PCのIPアドレス:8080 にアクセス
```

**ホスティングする場合：**
GitHub Pages、Netlify、Vercel等にデプロイ

### 2. ホーム画面に追加（おすすめ）

**iPhone：**
1. Safariでアプリを開く
2. 共有ボタン（□↑）をタップ
3. 「ホーム画面に追加」をタップ

**Android：**
1. Chromeでアプリを開く
2. メニュー（⋮）をタップ
3. 「ホーム画面に追加」をタップ

### 3. APIキーの設定（リアルタイムデータ）

1. [Twelve Data](https://twelvedata.com/) で無料アカウント作成
2. APIキーを取得
3. アプリの「設定」タブでキーを入力

### 4. LINE通知の設定

1. [LINE Notify](https://notify-bot.line.me/) にアクセス
2. LINEアカウントでログイン
3. 「トークンを発行する」をクリック
4. トークン名を入力（例：ゴールドシグナル）
5. 通知先を「1:1でLINE Notifyから通知を受け取る」に設定
6. 発行されたトークンをアプリの設定画面に貼り付け

## 画面説明

### シグナルタブ
- 現在のシグナル（買い/売り/様子見）
- 推奨エントリー価格、SL、TP
- 各インジケーターの状態

### チャートタブ
- ローソク足チャート
- EMA20（青）とEMA50（黄）
- ピンチでズーム、スワイプでスクロール

### 履歴タブ
- 過去のシグナル一覧
- 「未定」ボタンをタップで結果を記録（勝ち→負け→未定）

### 設定タブ
- ブラウザ通知のON/OFF
- LINE通知のON/OFF
- APIキーの設定

## シグナルの見方

| シグナル | 意味 |
|---------|------|
| 📈 買い | 買いエントリーのタイミング |
| 📉 売り | 売りエントリーのタイミング |
| ⏳ 様子見 | エントリーは控える |

## SL/TPの計算方法

- **SL（ストップロス）**: ATR × 1.5
- **TP（テイクプロフィット）**: ATR × 3
- **リスクリワード比**: 1:2

ATR（Average True Range）はボラティリティ指標で、相場の変動幅に応じてSL/TPが自動調整されます。

## トラブルシューティング

### LINE通知が届かない
- トークンが正しいか確認
- 設定画面でLINE通知がONになっているか確認
- 「テスト送信」ボタンで動作確認

### チャートが表示されない
- インターネット接続を確認
- ブラウザを最新版に更新

### データが更新されない
- APIキーが正しいか確認
- Twelve Dataの1日の制限（800回）を超えていないか確認
